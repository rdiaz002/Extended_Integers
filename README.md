# Extended_Integers
A library for Large Integer Values

## Getting started 
When initializing a extended integer input must be a ```char``` array.
```C++
ex_num a("100940394039");
```
or 
```C++
char b[] = "1090029020";
ex_num a(b);
```

**Note**
> No Negative numbers allowed. Negative Numbers will be included in future versions.
